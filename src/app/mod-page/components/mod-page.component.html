O que são Módulos?

Este tópico pode ajudá-lo a entender sobre módulos do Angular.

NgModules são contêineres para um bloco coeso de código dedicado a um domínio de aplicativo, um fluxo de trabalho ou um conjunto de recursos intimamente relacionado. Eles podem conter componentes, provedores de serviços e outros arquivos de código cujo escopo é definido pelo NgModule que o contém. Eles podem importar funcionalidades que são exportadas de outros NgModules e exportar funcionalidades selecionadas para uso por outros NgModules.

Metadados NgModule:


Metadados Descrição
declarations Os componentes , directivas e pipes que pertencem a este NgModule.
exports O subconjunto de declarações que devem ser visíveis e utilizáveis ​​nos modelos de componentes de outros NgModules.
imports Outros módulos cujas classes exportadas são necessárias para modelos de componentes declarados neste NgModule.
providers Criadores de serviços com os quais este NgModule contribui para a coleção global de serviços; eles se tornam acessíveis em todas as partes do aplicativo. (Você também pode especificar fornecedores no nível do componente.)
bootstrap A visualização principal do aplicativo, chamada de componente raiz , que hospeda todas as outras visualizações do aplicativo. Apenas o NgModule raiz deve definir a bootstrap propriedade.
