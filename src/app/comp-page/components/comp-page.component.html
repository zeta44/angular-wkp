O que são Componentes?

Este tópico pode ajudá-lo a entender sobre os componentes do Angular.

Os componentes são os blocos de construção que compõem um aplicativo. Um componente inclui uma classe TypeScript com um @Component() decorador, um modelo HTML e estilos.

O @Component() decorador especifica as seguintes informações específicas:


  • Um seletor CSS que define como o componente é usado em um modelo. Os elementos HTML em seu modelo que correspondem a este seletor tornam-se instâncias do componente.
  • Um modelo HTML que instrui o Angular sobre como renderizar o componente.
  • Um conjunto opcional de estilos CSS que definem a aparência dos elementos HTML do modelo.

O modelo de componente do Angular oferece forte encapsulamento e uma estrutura de aplicativo intuitiva. Os componentes também tornam seu aplicativo fácil de testar e podem melhorar a legibilidade geral de seu código. Para obter mais informações sobre o que fazer com os componentes, consulte a documentação oficial neste link: Componentes.
